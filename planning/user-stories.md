# MACROmate User Stories

## My Pantry

- As a user, I want to be able to save commonly eaten foods to my pantry so I can easily reuse them to plan meals.

- As a user, I want to be able to remove items from my pantry so I can eliminate clutter from unused foods.

## Today's Meals

- As a user, I want to add food from my pantry to today's meals so I can plan how to meet my daily macro intake.

- As a user, I want to adjust food quantities so I can keep my daily intake totals accurate.

- As a user, I want to remove foods from today's meals so I can correct errors and reflect meals not eaten.

## Macro Tracking

- As a user, I want to set my individual macro targets so I can meet my goals.

- As a user, I want to see remaining calories and macros so I know what I can still eat.

- As a user, I want to see the maximum allowable amount of each pantry food so I can more easily stick to my targets.

## Usability

- As a user, I want daily totals to update immediately so meal planning is quick and easy.

- As a user, I want the app to work on mobile screens so I can use it wherever I am preparing meals.
